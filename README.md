# StockLogic: Backend Inventory Management System
**2026 | Ongoing | CLI Application**

## Purpose
StockLogic automates stock tracking for small-scale warehouses to prevent manual entry errors and stock-out issues.

## Features
* **Full CRUD Operations**: Manage product lifecycles (Entry, Update, Delete).
* **Low-Stock Notification Engine**: Automated terminal alerts when quantities fall below a defined threshold.
* **Formatted Reports**: Clean data visualization using `console.table`.

## Tech Stack
* **Runtime**: Node.js
* **Database**: MySQL
* **Tools**: Git for Version Control
